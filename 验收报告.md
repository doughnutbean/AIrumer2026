# 可解释的谣言检测 — 项目验收报告

## 一、项目概述

本项目的目标是在社交媒体推文数据集上构建一个**可解释的谣言检测模型**。模型接收一条推文，输出两个结果：

1. **二分类标签**：0（非谣言）或 1（谣言）
2. **判断依据**：一段中文文字，解释模型为什么做出该判断

项目采用**复合模型架构**，将深度学习模型、大语言模型和检索增强生成（RAG）技术融合在一个流水线中，同时保证了检测性能和可解释性。

---

## 二、数据集

| 数据集 | 样本数 | 谣言(1) | 非谣言(0) | 事件类别 |
|--------|:-----:|:-------:|:---------:|:--------:|
| 训练集 train.csv | 2840 | 1240 (43.7%) | 1600 (56.3%) | 7 类 |
| 验证集 val.csv | 401 | 175 (43.6%) | 226 (56.4%) | 7 类 |

数据集为英文推文，涵盖多个社会事件主题（Ferguson事件、Gurlitt艺术品等）。类别分布基本均衡。

## 三、模型架构

### 3.1 整体流水线

```
输入推文文本
    │
    ├─→ [BiGRU 谣言检测] ──→ label (0/1) + 置信度
    │
    ├─→ [RAG 检索增强] ──→ 检索训练集相似样本
    │
    └─→ [LLM 判断依据生成] ──→ 结构化中文分析
                             (SJTU API · deepseek-chat)
    │
    └─→ 输出: {label, prediction, probability, judgment_basis}
```

### 3.2 组件一：BiGRU 深度学习检测器

| 项目 | 说明 |
|------|------|
| 模型 | 2 层双向 GRU + 加性注意力 + 事件嵌入 |
| 词嵌入 | GloVe 6B 300d 预训练词向量（词表命中率 96.0%） |
| 参数量 | 746,429 |
| 输入 | 推文文本 → 分词 → 编码为索引序列 (max_len=64) |
| 输出 | 二分类 logits → Sigmoid → 概率值 |

### 3.3 组件二：RAG 检索增强

| 项目 | 说明 |
|------|------|
| 检索方法 | TF-IDF 向量化 + 余弦相似度 |
| 向量维度 | 5000 维 (unigram + bigram) |
| 检索数量 | Top-3 最相似样本 |
| 用途 | 将相似样本的标签和内容注入 LLM prompt，增强判断准确性 |

### 3.4 组件三：LLM 判断依据生成

| 项目 | 说明 |
|------|------|
| 接口 | 上海交通大学本地大模型 API（OpenAI 兼容格式） |
| 模型 | `deepseek-chat`（DeepSeek V3.2） |
| 调用方式 | System Prompt + User Message |
| 分析维度 | 文本内容、语言风格、信息来源、与已知样本对比 |

## 四、实验结果

### 4.1 检测性能（val.csv）

| 指标 | 取值 |
|------|:----:|
| **Accuracy** | **0.8603** |
| **Precision** | **0.8563**|
| **Recall** | **0.8171** |
| **F1 Score** | **0.8363** |

按类别详细报告：

| 类别 | precision | recall | f1-score | support |
|------|:---------:|:------:|:--------:|:-------:|
| 非谣言(0) | 0.86 | 0.89 | 0.88 | 226 |
| 谣言(1)   | 0.86 | 0.82 | 0.84 | 175 |

### 4.2 判断依据质量示例

**输入推文：**
> BREAKING: Ferguson police chief just announced that officer Darren Wilson shot the unarmed teen, Michael Brown.

**BiGRU 检测：** 谣言 (98.37%)

**LLM 判断依据：**
> 1. **信息来源不可靠**：推文以"BREAKING"开头制造紧迫感，但未提供任何具体来源（如警方官方声明、新闻机构报道），缺乏可验证的引用，符合谣言常见的信息模糊特征。
> 2. **与已知谣言高度相似**：训练集中相似样本均标注为谣言，高度重复的措辞模式是谣言扩散的典型手法。
> 3. **语言煽情化**："unarmed teen"刻意强化情感冲击，引导不经核实转发。
> 4. **缺乏关键细节**：未包含具体时间、地点、发言人姓名，信息残缺。

## 五、功能说明

```bash
# 1. 配置 API Key（在 config.py 中填入）
# LLM_API_KEY = "your_sjtu_api_key"

# 2. 运行测试
python test_composite.py
```

## 七、创新点

1. **复合模型架构**：将传统的深度学习检测模型与现代大语言模型结合，兼顾检测精度与可解释性。
2. **RAG 检索增强**：通过检索训练集中的相似样本，为 LLM 提供参考案例，增强判断依据的准确性。
3. **预训练词向量**：使用 GloVe 300d 预训练词向量，词表命中率 96%，大幅提升模型语义理解能力。
4. **多维度推理**：LLM 从内容、来源、语言风格、相似案例等维度综合判断，输出的依据结构化、易理解。
